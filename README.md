# About

Managed dotfiles & Install some softwares on multi-environments.

# Execute

* download

```bash
git clone https://github.com/ShotaKitazawa/dotfiles $HOME/dotfiles
```

* deploy dotfiles

```bash
./dotfiles/put_profile
```

* source bashrc

```bash
. ~/.bashrc_self
```

* install applications

```bash
./dotfiles/install_packages
```
