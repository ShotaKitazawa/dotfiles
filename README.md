# Usage

* download
```bash
git clone https://github.com/ShotaKitazawa/dotfiles $HOME/dotfiles
```

* check tmux version & link to .tmux.conf
```bash
cd $HOME/dotfiles
ln -s tmux-before2.4/.tmux.conf .
```

* deploy
```bash
./dotfiles/putprofile
```
